 Task Management Application

 Overview

Task Management Application is a full-stack web application that allows users to create, manage, update, and track tasks efficiently. The application provides user authentication, task management functionality, and a responsive user interface for a seamless user experience.

 Features

User Authentication

* User Registration
* User Login
* User Logout
* Secure access to task management features

Task Management (CRUD Operations)

* Create new tasks
* View existing tasks
* Update task details
* Delete tasks

Dashboard

* Display all tasks
* Add new tasks from the dashboard
* Edit existing tasks
* Remove completed or unwanted tasks

 

Technologies Used
 Frontend

* HTML
* CSS
* JavaScript

Backend
* Node.js
* Express.js

Database
* MySQL
